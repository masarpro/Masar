**Masar** is an all-in-one cloud platform for managing construction projects, built for contractors and engineering offices in Saudi Arabia. It brings everything you need into one place: projects, e-invoicing, expenses, estimation and quantity takeoff, subcontractors, employees and payroll, and an owner portal.

## Where do I start?

If you're new to Masar, start with the [Getting Started](/docs/getting-started) guide — within minutes you'll have your account and first project set up.

For quick questions, try the [FAQ page](/faq) or ask the **Masar AI assistant** from the chat icon on the website.
