## محاسبة بلا مجهود

النظام المحاسبي في مسار يعمل **تلقائياً دائماً** — لا يحتاج تفعيلاً ولا إعداداً:

- مع أول عملية مالية يُنشأ **دليل حسابات** جاهز للمقاولات (النقدية، العملاء، الموردون، إيرادات المشاريع، مقاولو الباطن...).
- كل عملية (فاتورة، مصروف، دفعة، رواتب، تحويل...) تولّد **قيداً محاسبياً تلقائياً** مرحّلاً.
- أخطاء القيود التلقائية — إن حدثت — لا توقف عمليتك المالية أبداً.

## دليل الحسابات

هرمي وقابل للتخصيص من **المالية ← دليل الحسابات**: أضف حسابات فرعية، وعدّل المسميات، مع أرصدة افتتاحية لبداية استخدامك للنظام.

## القيود اليومية

- **تلقائية:** لكل عملية مالية قيد بترقيم مرجعي واضح (مثل `INV-JE-2026-0001` لقيود الفواتير).
- **يدوية:** أنشئ قيوداً يدوية لأي تسويات، مع دورة ترحيل.
- **متكررة:** قوالب قيود شهرية (إيجار، إهلاك...) تتولد تلقائياً.

## التقارير المحاسبية

من **المالية ← التقارير المحاسبية**:

- ميزان المراجعة
- قائمة الدخل
- الميزانية العمومية
- دفتر الأستاذ وكشوف الحسابات
- مراكز التكلفة (لكل مشروع)
- تقرير ضريبة القيمة المضافة التفصيلي

كلها قابلة للطباعة والتصدير إلى Excel.

## الفترات المحاسبية والإقفال

- أقفل الفترات المحاسبية المنتهية لمنع أي تعديل بأثر رجعي.
- نفّذ **الإقفال السنوي** لترحيل الأرباح المبقاة وبدء سنة مالية جديدة.

## حسابات الشركاء

تابع **مساهمات رأس المال** و**مسحوبات الشركاء** لكل شريك، مع قيود تلقائية وتقارير موقف لكل شريك.
