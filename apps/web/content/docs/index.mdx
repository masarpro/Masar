**مسار** منصة سحابية متكاملة لإدارة المشاريع الإنشائية، مبنية للمقاولين والمكاتب الهندسية في السعودية. تجمع في مكان واحد كل ما تحتاجه لإدارة أعمالك: المشاريع، الفوترة الإلكترونية، المصروفات، التسعير وحساب الكميات، مقاولي الباطن، الموظفين والرواتب، وبوابة المالك.

## أقسام التوثيق

| القسم | ماذا ستتعلم |
|-------|-------------|
| [البدء مع مسار](/docs/getting-started) | إنشاء الحساب، إعداد المنشأة، ودعوة فريقك |
| [إدارة المشاريع](/docs/projects) | إنشاء المشاريع، متابعة التنفيذ، والتقارير اليومية |
| [التسعير وحساب الكميات](/docs/pricing-studies) | دراسات التكلفة، حساب الكميات الإنشائية، وعروض الأسعار |
| [الفوترة والمالية](/docs/finance) | الفواتير المتوافقة مع زاتكا، المصروفات، والسندات |
| [المحاسبة](/docs/accounting) | القيود التلقائية، دليل الحسابات، والتقارير المالية |
| [مقاولو الباطن](/docs/subcontractors) | العقود، المستخلصات، الدفعات، والمحتجزات |
| [الموظفون والرواتب](/docs/company-hr) | ملفات الموظفين، مسير الرواتب، والإجازات |
| [بوابة المالك](/docs/owner-portal) | مشاركة تقدم المشروع مع المالك برابط آمن |
| [المساعد الذكي](/docs/ai-assistant) | كيف تسأل المساعد عن بياناتك وتحصل على إجابات فورية |
| [الخطط والاشتراكات](/docs/plans-billing) | الأسعار، الترقية، وإدارة اشتراكك |

## من أين أبدأ؟

إذا كنت جديداً على مسار، ابدأ بدليل [البدء مع مسار](/docs/getting-started) — خلال دقائق ستنشئ حسابك ومشروعك الأول.

ولأي سؤال سريع، جرّب [صفحة الأسئلة الشائعة](/faq) أو اسأل **مساعد مسار الذكي** من أيقونة المحادثة في الموقع.
